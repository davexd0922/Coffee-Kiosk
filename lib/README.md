# Library Folder

Place optional local jars here, such as the PostgreSQL JDBC driver.

Recommended setup for NetBeans:

1. Right-click project > Properties.
2. Open Libraries.
3. Add the PostgreSQL JDBC jar.
4. Add JavaFX SDK jars if your JDK does not include JavaFX.

The source code compiles against standard JDBC interfaces, but the PostgreSQL driver jar is required at runtime.
